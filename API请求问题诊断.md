# API请求问题诊断 - 只显示1-2个股票

## ❌ 问题：只显示1-2个股票（不正常）

### 应该显示多少个？

**配置**：5个股票
**预期**：应该显示5个股票（AAPL, MSFT, NVDA, GOOGL, GOOG）

### 可能的原因

#### 1. API速率限制（最可能）

**问题**：
- 代码同时发送5个API请求（并行）
- 但API限制是：**每分钟5次请求**
- 如果之前的请求还没"过期"，新请求可能被拒绝

**检查方法**：
- 打开浏览器开发者工具（F12）
- 查看Console标签，看有没有错误信息
- 查看Network标签，看有多少API请求失败

#### 2. 请求间隔问题

即使限制是每分钟5次，如果5个请求同时发送：
- 可能只有前几个成功
- 后面的请求可能因为"同一分钟内的请求过多"而被拒绝

#### 3. 等待时间不够

如果你刚刚刷新过：
- 需要等待1分钟让限制重置
- 然后再刷新
