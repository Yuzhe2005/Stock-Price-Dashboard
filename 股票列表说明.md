## 如何修改股票列表？

### 方法 1：修改代码（推荐）

1. 打开 `src/services/stockAPI.ts` 文件
2. 找到第 10 行：
   ```typescript
   const DEMO_STOCKS = ['AAPL', 'GOOGL', 'MSFT', 'AMZN', 'TSLA', 'META', 'NVDA', 'NFLX'];
   ```
3. 修改为你想要的股票代码，例如：
   ```typescript
   const DEMO_STOCKS = ['AAPL', 'MSFT', 'GOOGL'];  // 只显示3个
   ```

   ```typescript
   const DEMO_STOCKS = ['AAPL', 'GOOGL', 'MSFT', 'AMZN', 'TSLA', 'META', 'NVDA', 'NFLX', 'DIS', 'JPM'];  // 添加更多
   ```

### 注意事项

1. **股票代码必须是有效的**：Alpha Vantage API 需要支持该股票代码
2. **美股代码**：通常是大写字母，1-5个字符
3. **API 限制**：免费版每天500次请求，如果显示8个股票，刷新50次就会用完
4. **修改后需要重启**：修改代码后，需要重启开发服务器（`npm run dev`）
